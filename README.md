# textTip

テキストに説明文をつけることをが可能です。
タグは考慮されておりません。

```
yarn add texttip 
or 
npm install texttip


texttip('ID name',jsonData);
```

===
jsonData 例
===
```
[
 {
  "id":1,
  "title":"認識",
  "disc":"ある物事を知り、その本質・意義などを理解すること。また、そういう心の働き。「―が甘い」「―を新たにする」「―を深める」「対象を―する」"
 },
  {
  "id":2,
  "title":"覚醒",
  "disc":"目を覚ますこと。目が覚めること。「昏睡状態から―する」"
 },
 ...
]
```
